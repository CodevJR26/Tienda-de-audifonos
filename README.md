# Audifonos
El sitio web TechPRO fue concebido y desarrollado bajo un enfoque de diseño web moderno, escalable y centrado en la experiencia del usuario, aplicando buenas prácticas de UI/UX y una arquitectura visual clara.

La estructura general se apoya en el uso de CSS Grid y Flexbox, lo que permite una composición adaptable y eficiente en distintos tamaños de pantalla. Desde el header, se establece una jerarquía visual fuerte mediante contraste cromático, tipografías de alto peso visual y un diseño dividido que destaca el producto principal y su propuesta de valor.

La identidad visual se gestiona mediante custom properties (CSS variables), asegurando consistencia en colores, tipografías y espaciados, además de facilitar el mantenimiento y la escalabilidad del proyecto. La paleta cromática refuerza el concepto tecnológico de la marca, utilizando el color primario para llamadas de atención y elementos clave, mientras que los tonos neutros equilibran la composición.

Las secciones informativas, como los beneficios del producto, priorizan la lectura rápida y la claridad del mensaje, utilizando iconografía y alineación centrada para una comunicación directa. En la sección Sobre TechPRO, se implementa una superposición de imágenes y gradientes que permite integrar contenido visual sin comprometer la legibilidad del texto.

El catálogo de modelos se diseñó con una jerarquía visual clara, destacando el producto principal mediante mayor presencia espacial, color y tamaño. Se incorporaron transiciones y efectos hover sutiles para mejorar la interacción del usuario sin afectar el rendimiento ni la accesibilidad.

La sección de newsletter cumple una función estratégica dentro del flujo del sitio, integrándose visualmente al diseño mediante fondos combinados y formularios simples, optimizados para reducir fricción y favorecer la conversión.

En su conjunto, TechPRO representa un proyecto enfocado en claridad visual, consistencia estética y funcionalidad, demostrando la correcta aplicación de diseño responsivo, organización del código CSS y principios de diseño orientados a producto y negocio.
